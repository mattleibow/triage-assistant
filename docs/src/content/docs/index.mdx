## Quick Example

```yaml
name: 'AI Triage'
on:
  issues:
    types: [opened]
  pull_request:
    types: [opened]

jobs:
  triage:
    runs-on: ubuntu-latest
    permissions:
      contents: read
      issues: write
      pull-requests: write
      models: read
    steps:
      - uses: actions/checkout@v5
      - uses: mattleibow/triage-assistant/apply-labels@v1
        with:
          apply-labels: true
          apply-comment: true
```
